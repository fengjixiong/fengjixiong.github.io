Projects & Researches

  1. Named Entity Recognition based on Web Information.  (2017)
    The project included fetching the web pages, automatically analysing texts and displaying the results, where spider techniques, deep learning methods and web front-end programming were used.
  2. Building a specific 3D Engine.  (2016)
    The project was designed to develop a specific 3D engine for transportation. The 2D diagrams in AutoCAD could be automatically transformed into 3D scenes. This project will be used in VR as a extension in the future.
  3. Research on Handwritten Historical Chinese Character Recognition.  (2013-2015) [detail]
    This research proposed a non-linear style transfer mapping based on Gaussian process (GP-STM), to recognize the handwritten historical Chinese characters with a insufficient training set.
  4. Research on Gait based Human Recognition. (2012-2013) [detail]
    This research proposed a HMM based method to recognize humans from their gait videos in a limited circumstance.
  5. A Hand-ring System with Pulse-Measurement and Locating Service. (2012-2013) [detail]
    This project was supported by the fund from the Zhejiang Province government. It aimed to build an on-hand-system, which was monitoring the user's pulse continuously. Once the user's pulse became abnormal, the system would send text message to the user's families.
  6. Embedded Visual IP Telephone. (2012)  [detail]
    The project was a branch of the smart home system. It was required to be embedded to be fixed on the houses' doors. Protocols like SIP (Session Initiation Protocol), RTP (Real-time Transport Protocol) and RTCP (Real-Time Control Protocol) were implemented on the DSP and ARM processors.
