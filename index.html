Projects & Researches

  1. Project of Image Server for the very Large TIFF.  (2017)
    The project utilized tens of server computers to provide TIFF images for the users. TIFF images with sizes over 10GB were delivered on the internet by small JPG tiles, where image pyramids and Nginx were used.
  2. Porject of Data Mining base on Web Information.  (2016)
    The project included fetching the web pages, automatically analysing texts and displaying the results, where the spider, TensorFlow-CNN-model and web front-end techniques were used.
  3. Project of Transferring Oracle to Distribute Data Warehouse.  (2015)
    The project aimed to transfer the data in Oracle to distribute data warehouse, for the purpose of data mining and larger capacity. Apache Hawq was used here.
  4. Project of Building a specific 3D Engine.  (2015)
    The project was designed to develope a specific 3D engine for railway. The 2D diagram in AutoCAD could be automatically transformed into 3D scene. The project could be easily used in VR.
  5. Research on Handwritten Historical Chinese Character Recognition.  (2013-2015)
    This research proposed a non-linear style transfer mapping based on Gaussian process (GP-STM), to recognize the handwritten historical Chinese characters with a insufficient training set.
  6. Research on Gait based Human Recognition. (2012-2013)
    This research proposed a HMM based method to recognize humans from their gait videos.
  7. Research on A Locating Handring System base on GPS. (2012-2013)
    This research was supported by Zhejiang Province government. The research aimed to build an on-hand-system, which monitored the user’s pulse.Once the user’s pulse became abnormal, the system would send text message to the user’s families.
